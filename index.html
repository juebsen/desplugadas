O pensamento computacional pode ser definido como uma habilidade para resolver problemas e desafios de forma eficiente, assim como um computador o faria. Essa resolução pode ou não envolver equipamentos tecnológicos, mas a sua base é a exploração de forma criativa, crítica e estratégica dos domínios computacionais. Pixel Art: É uma técnica de desenho que consiste em criar imagens a partir de pequenos quadrados coloridos, como se fossem pixels mesmo. E uma forma de arte digital que utiliza pequenos quadrados coloridos para criar imagens. É uma atividade educativa e lúdica, pois ajuda a desenvolver habilidades motoras finas e a criatividade. Os desenhos de Pixel Art variam desde personagens de desenhos animados até animais e objetos do cotidiano. Para colorir, os estudantes podem usar lápis de cor, canetinhas ou tintas.
